Anleitung
So bekommst du die Tokens
1. Öffne die Google Cloud Console und erstelle ein Projekt.
2. Aktiviere die Google Drive API unter „APIs & Services".
3. Erstelle unter „Anmeldedaten" eine OAuth 2.0 Client-ID (Typ: Desktop-App).
4. Kopiere Client ID und Client Secret aus den Anmeldedaten.
5. Öffne den OAuth 2.0 Playground. Klicke auf ⚙️ und aktiviere „Use your own OAuth credentials". Gib Client ID & Secret ein.
6. Wähle den Scope https://www.googleapis.com/auth/drive.file und klicke „Authorize APIs".
7. Klicke „Exchange authorization code for tokens" und kopiere den Refresh Token.
